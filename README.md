# 💻 Java Mini Projects

This repository contains two Java console-based applications developed to strengthen core Java concepts and problem-solving skills.

---

## 📌 Projects Included

### 1️⃣ ATM Interface 🏧
A simple ATM simulation system where users can perform basic banking operations.

#### 🔹 Features
- User login using account number and PIN
- Check account balance
- Deposit money
- Withdraw money
- Input validation for secure transactions

#### 🔹 Sample Accounts
| Account No | PIN  | Balance |
|-----------|------|--------|
| 1001      | 1234 | ₹5000  |
| 1002      | 4321 | ₹2500  |

#### 🔹 Concepts Used
- Classes & Objects
- HashMap (Data Storage)
- Conditional Statements
- User Input Handling

---

### 2️⃣ Number Guessing Game 🎯
An interactive game where the system generates a random number and the user tries to guess it.

#### 🔹 Features
- Random number generation (1–100)
- User input validation
- Hints like *Too High* / *Too Low*
- Attempt counter
- Replay option

#### 🔹 Concepts Used
- Random Class
- Loops & Conditions
- Scanner Class
- Input Validation

---

## ⚙️ Technologies Used
- Java (Core Java)
- VS Code / Any Java IDE
- Command Line Interface (CLI)
